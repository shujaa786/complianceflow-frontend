
# ComplianceFlow AI SaaS Application
  
ComplianceFlow AI is an AI-powered enterprise document workflow and compliance management platform that enables organizations to:
upload business documents,
extract structured metadata using AI,
manage approval workflows,
track compliance activities,
maintain audit logs,
generate analytics,
and automate document operations securely.

### ✨ Key Features

- **Login Screen** — Enterprise authentication with email/password and Google SSO option
- **Main Dashboard** — KPI cards, approval trends chart, document distribution pie chart, and recent activity timeline
- **Document Upload** — Drag-and-drop interface with real-time upload progress and AI processing status
- **AI Metadata Extraction** — Document preview with editable extracted fields, confidence scores, and validation warnings
- **Document Management** — Comprehensive table with advanced filters, search, sorting, and status badges
- **Approval Workflow** — Multi-step timeline, document preview, comments, assigned reviewers, and SLA tracking
- **Audit Logs** — Detailed activity timeline with user actions, timestamps, IP addresses, and export functionality
- **Notifications Center** — Real-time alerts with configurable settings for approvals, compliance, and expiry reminders
- **Analytics & Reporting** — Charts for approval trends, department performance, compliance issues, and processing times
- **Admin Settings** — User management, department configuration, workflow settings, and integrations panel

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
