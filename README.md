
# ComplianceFlow AI SaaS Application
  
ComplianceFlow AI is an AI-powered enterprise document workflow and compliance management platform that enables organizations to:
upload business documents,
extract structured metadata using AI,
manage approval workflows,
track compliance activities,
maintain audit logs,
generate analytics,
and automate document operations securely.

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
